# SoftDef Frontend Test

This project is created as part of the Frontend Developer Internship assignment.

The goal of this project is to recreate the provided Figma design using React/Next.js and Tailwind CSS while keeping the layout responsive and visually close to the original design.

## Tech Stack

* Next.js
* React.js
* Tailwind CSS

## Features

* Responsive design
* Reusable components
* Mobile navigation
* Product sections
* Customer review section
* Footer with newsletter form
* Hover effects and transitions

## Project Setup

Clone the repository:

```bash
git remote add origin https://github.com/vivekx11/softdef-frontend-test.git
```

  

  

Move into the project folder:

```bash
cd softdef-frontend-test
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```

## Deployment

The project is deployed using render

## Notes

* The UI is built based on the provided Figma design.
* Tailwind CSS is used for styling and responsiveness.
* Components are separated for better structure and readability.
